HOW TO SETUP GAME CORRECTLY
1.	Switch to Android or iOS platform
2.	Download and install SPINE runtime for UNITY: https://drive.google.com/open?id=17HrP3ecQ7yl31MluwvV2SvvMQCgH2ixt
3.	Now you can play game by open Logo scene in TD/Scene/ folder
4.	If wanna use IAP, Turn on IAP: https://docs.unity3d.com/Manual/UnityIAPSettingUp.html
